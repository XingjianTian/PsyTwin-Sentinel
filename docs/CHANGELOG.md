## [Unreleased]

### Added - OpenClaw 编排中心 (AI Orchestration Dashboard)

#### 智能体集群可视化
- 田字格 3×3 网格布局展示 7 个智能体节点实时移动
- 每个智能体拥有独特的颜色主题和样式标识
- 平滑的网格路径动画，模拟智能体在网格中的流转
- 组件：`AgentGridOffice`（容器）+ `AgentGridLabel`（移动卡片）

#### 实时数据流 (WebSocket → SSE 桥接)
- Server-Sent Events (SSE) 实现服务器到客户端的实时推送
- 模拟智能体工作流数据流（请求流转、状态更新）
- 实时统计数据更新（Token 消耗、请求数、响应时间）

#### 请求流转追踪
- 进度条可视化展示请求生命周期
- 实时追踪请求从发起到完成的各个阶段
- 最近 5 条请求历史记录展示

#### 活动日志面板 (LivePanel)
- 合并的请求/活动统一面板
- Tab 切换：进行中请求 vs 活动日志
- 实时滚动更新，最多显示 5 条最新记录
- 时间戳、状态标识、详细信息展示

#### AI 思维展示 (TeamDashboard)
- 每个智能体的"思绪气泡"轮播展示
- 模拟 AI 处理任务时的内部思考过程
- 动态切换不同智能体的思维片段

#### 成本分析仪表盘 (CostDashboard)
- Token 消耗统计（输入/输出/总计）
- AI vs 人力成本对比可视化
- Agent 节省排行榜（按节省金额排序）
- 成本趋势图表（日/周/月视图）

#### 安全审计中心 (SecurityDashboard)
- 系统健康度仪表盘（实时评分）
- 端口扫描状态监控
- 安全事件日志展示
- 健康度趋势图表

#### 数据库浏览看板 (DatabaseDashboard)
- openclaw_* 系列表结构浏览
- 分页查看表数据（每页 10 条）
- 表字段信息展示（类型、约束等）
- 支持表切换和数据预览

#### 服务端 API
- `GET /api/openclaw/config` - 获取智能体配置和样式映射
- `GET /api/openclaw/stats` - 获取实时统计数据
- `GET /api/openclaw/stats/agents` - 获取智能体效率排行榜
- `GET /api/openclaw/database` - 浏览数据库表结构和数据
- `GET /api/openclaw/workflow` - 获取工作流执行数据

### Changed

#### UI/UX 改进
- 侧边栏宽度从 224px (`w-56`) 拓宽至 256px (`w-64`)
- OpenClaw 编排 Tab 按钮改为卡片样式展示
- 右侧面板合并：请求追踪和活动日志统一为 LivePanel
- 新增图标：Brain, Workflow, Shield, Database, Coins, Activity
