### Added

- **干预记录详情功能 (M05)** - 完整的详情弹窗与数据结构
  - 详情弹窗组件: 4个Tab分页展示干预全流程数据
  - 数据库架构: 独立 `InterventionDetail` 表存储详细干预数据
  - 干预前评估: 情绪状态(焦虑/抑郁/压力)、问题描述、风险评估
  - 干预过程: 会话内容、干预技术、学生反馈、关键要点
  - 干预效果: 前后对比、改善评分、关键突破、未解决问题
  - 后续建议: 行动计划、下次预约、转介建议、咨询师笔记
  - 详情按钮绑定: 点击后异步加载并展示完整干预数据
  - 数据填充脚本: `scripts/seed-intervention-details.ts`
- **小程序数据库对齐** - 扩展 Schema 支持 PsyTwin Pocket
  - 新增枚举: StudentStatus, CommentStatus, MessageStatus, SessionStatus
  - 扩展模型: Student, Post, Comment, ChatSession, ChatMessage, Appointment
  - 新增 API 契约文档 (docs/api_contract.md)
- **学生心理档案模块 (M03)** - 完整功能实现
  - 学生列表页: 搜索/筛选/分页
  - 学生详情页: 心理画像雷达图/生命周期/干预记录
  - OpenSpecs 详细设计规范
- **Phase 3 AI 集成** - Qwen API + RAG 向量存储
  - AI 风险评估报告自动生成
  - AI 配置页面 UI + API
- **Next.js App Router 路由架构重构**
  - 物理路由: /dashboard, /risk-trace, /students 等
- **开发环境优化**
  - PostgreSQL + pgAdmin Docker 配置
