## [Unreleased]

### Added

- **多模态数据流实时监控** - 新增实时生理曲线展示（心率、血氧、HRV）
  - SSE endpoint: `GET /api/multimodal/sensors/stream` - 通过 Redis Pub/Sub 实时推送数据
  - 学生列表 API: `GET /api/multimodal/students` - 显示所有学生及最新生理数据
  - 前端图表支持历史数据从左到右展开动画
- **语音流实时转写** - 基于百度 ASR 的语音识别（通过 PsyTwin-Companion WebSocket-Server）
  - WebSocket `/ws/audio` 接收音频流
  - 语音活动检测 (VAD) 自动分段
  - 实时转写显示在语音流卡片中
- **学生端通知 API** - 完整的通知系统
  - `GET /api/pocket/student/my/notifications` - 获取通知列表（支持分页和筛选）
  - `PUT /api/pocket/student/my/notifications/:id/read` - 标记通知已读
  - `POST /api/pocket/student/my/notifications` - 发送通知（供 Sentinel 内部调用）
  - 支持通知类型：system, appointment, warning, chat, post, comment
- **语音波形可视化** - 实时音频电平显示
  - 基于 `voice_level` SSE 事件
  - 彩色波形图区分高/中/平稳唤醒区
- **视觉流表情识别** - 多语言支持
  - 支持 happy, sad, angry, fear, surprise, disgust, neutral 等表情
  - 大图标 emoji 显示（5xl）
  - 中文描述（开心、悲伤、愤怒等）
