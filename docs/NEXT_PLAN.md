### Phase 2: 多模态数据流实时监控（核心亮点）

#### P2.1 实时数据流监控页面

**入口**: 从心理咨询室点击"查看实时数据"

| 数据流 | 展示内容 | 可视化方式 |
|--------|----------|------------|
| 视觉流 | 微表情热图、眼动轨迹、瞳孔变化 | 实时热力图、轨迹动画 |
| 语音流 | 情感倾向、语音颤抖指数、语义标签 | 实时波形图 + 情感色块 |
| 生理流 | 心率、HRV、压力指数、Alpha/Beta/Theta波 | 多曲线实时图表 |
| 交互流 | 操作频率、手部震颤、决策延迟、回避次数 | 事件流 + 数值卡片 |

**页面结构**:
```
├── 左侧：当前体验学生列表
│   ├── 姓名 + 头像
│   ├── 所在咨询室
│   ├── 当前状态（焦虑/平静/紧张...）
│   └── 风险等级标记
│
└── 右侧：选中学生的实时数据面板
    ├── 顶部：基本信息 + 风险等级
    ├── 视觉流模块（微表情 + 眼动）
    ├── 语音流模块（波形 + 情感分析）
    ├── 生理流模块（心率/HRV/GSR/脑电）
    └── 交互流模块（行为事件）
```

## 四、技术实现要点

### 多模态数据流

- WebSocket 实时推送
- 数据缓冲（最近60秒显示）
- 异常事件自动标记（红色闪烁）

### 心理咨询室预约

- 时段锁定设备
- 自动释放（超时提醒）
- 冲突检测
