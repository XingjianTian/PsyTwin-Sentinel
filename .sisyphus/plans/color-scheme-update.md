# PsyTwin-Sentinel 明亮配色方案设计

## 1. 设计目标

将当前「深色科技风」转换为「明亮温暖风格」，参考图片配色但适配心理健康主题。

---

## 2. 配色对比

### 当前配色（深色科技风）

| 用途 | 当前颜色 | Hex |
|------|----------|-----|
| 主背景 | 深海军蓝 | `#080c1a` |
| 卡片背景 | 深蓝灰 | `#0f1629` |
| 主强调色 | 青色 | `#00d4ff` |
| 次要色 | 橙色 | `#f97316` |
| 文字 | 浅灰 | `#e2e8f0` |
| 边框 | 深灰蓝 | `#1e2a45` |

### 目标配色（明亮温暖风）

| 用途 | 目标颜色 | Hex | 风格来源 |
|------|----------|-----|----------|
| 主背景 | 暖米白 | `#FAF8F5` | 参考图片 #FEF9F2 暖调 |
| 卡片背景 | 纯白 | `#FFFFFF` | 干净通透 |
| 主强调色 | 柔和蓝紫 | `#7C3AED` | 专业、信任感 |
| 次强调色 | 温暖绿 | `#10B981` | 健康、心理平静 |
| 风险红色 | 柔和红 | `#EF4444` | 警示但不刺眼 |
| 主文字 | 深灰 | `#1F2937` | 高可读性 |
| 辅助文字 | 中灰 | `#6B7280` | 柔和 |
| 边框 | 浅灰 | `#E5E7EB` | 轻盈 |

```
:root {
  /* ===== 主背景色 ===== */

  /* ===== 卡片组件 ===== */
  --card: #FFFFFF;                  /* 纯白，卡片背景 */
  --card-foreground: #1F2937;       /* 深灰，文字 */

  --popover: #FFFFFF;
  --popover-foreground: #1F2937;

  --primary: #7C3AED;               /* 柔和蓝紫，主按钮、链接 */
  --primary-foreground: #FFFFFF;    /* 白色，字 */

  /* ===== 次要色 ===== */
  --secondary: #F3F4F6;            /* 浅灰，次要背景 */

  --accent-foreground: #4F46E5;     /* 紫，字 */

  --destructive-foreground: #FFFFFF;

  --warning-foreground: #FFFFFF;

  /* ===== 图表色 ===== */

  --sidebar-foreground: #1F2937;

  --sidebar-primary-foreground: #FFFFFF;
  --sidebar-accent: #F3F4F6;
  --sidebar-accent-foreground: #374151;
  --sidebar-border: #E5E7EB;
  --sidebar-ring: #7C3AED;
}
```

### 3.2 风险等级配色（心理健康主题）

| 风险等级 | 背景色 | 文字色 | 风格 |
|----------|--------|--------|------|
| 红色预警 | `#FEF2F2` (淡红) | `#DC2626` | 柔和但不忽视 |
| 橙色预警 | `#FFF7ED` (淡橙) | `#EA580C` | 温暖提醒 |
| 黄色预警 | `#FFFBEB` (淡黄) | `#D97706` | 柔和警示 |
| 绿色安全 | `#F0FDF4` (淡绿) | `#16A34A` | 平静舒适 |

### 3.3 组件特定配色

#### 统计卡片 (StatCards)
- 背景：`#FFFFFF`
- 边框：`#E5E7EB`
- 图标背景：`#EEF2FF`（淡紫）
- 数字颜色：`#1F2937`

#### 热力图 (HeatmapCard)
- 高风险区域：`#FECACA` → `#EF4444`
- 中风险区域：`#FDE68A` → `#F59E0B`
- 低风险/安全：`#BBF7D0` → `#22C55E`
- 背景：`#FFFFFF`

#### 预警雷达 (AlertRadarCard)
- 雷达背景：`#FFFFFF`
- 扫描线：`#7C3AED` (主强调色)
- 预警点：`#EF4444`

#### 漏斗图 (FunnelCard)
- 阶段颜色渐变：`#7C3AED` → `#A78BFA` → `#C4B5FD`

## 4. 风格特点

### 4.1 与参考图片的对比

| 维度 | 参考图片 | 本方案 | 原因 |
|------|----------|--------|------|
| 主题 | 绿色能源/环保 | 心理健康/关怀 | 产品定位不同 |
| 主色 | #C6F16D 嫩绿 | #7C3AED 蓝紫 | 蓝紫代表专业、信任 |
| 辅助 | #06211C 深森林 | #10B981 温暖绿 | 绿色代表健康、平静 |
| 背景 | #FEF9F2 奶油色 | #FAF8F5 暖米白 | 相似的温暖调 |

### 4.2 设计原则

1. **温暖亲和**：避免深色带来的压抑感，使用暖米白背景
2. **专业可信**：蓝紫色主调，传达专业与信任
3. **健康平和**：绿色辅助色，呼应心理健康主题
4. **层次分明**：浅灰边框替代深色边框，轻盈通透
5. **数据清晰**：图表使用明亮色彩，对比度适中
